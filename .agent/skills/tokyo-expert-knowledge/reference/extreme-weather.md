# 極端天氣應對知識 (Extreme Weather Response)

> 本文件提供大雪、颱風等極端天氣事件的交通應對知識，供 AI Agent 在回答相關問題時參考。

---

## 大雪 (Heavy Snow) 應對指南

### 電車・地鐵停駛判斷邏輯

| 條件 | 可能影響 |
|------|----------|
| 積雪 20cm 以上 | JR 在來線開始減班或限速運行 |
| 積雪 30cm 以上 | 部分路線可能暫停運行 |
| 暴風雪警報發布 | 高機率大規模停駛 |
| 凌晨大雪 | 首班車可能延遲（需確認除雪作業完成） |

**例外**：
- **東京Metro / 都營地鐵**：全線地下運行，通常不受積雪影響，但地面段（如丸之內線荻窪方向）可能受影響。
- **新幹線**：東海道新幹線較易受影響（關原區間），北陸・上越新幹線有除雪設備，影響較小。

### 大雪時的替代交通建議

1. **優先使用地下鐵**：東京Metro、都營地下鐵全線地下化路段不受影響。
2. **避免使用山手線外側的 JR 線**：埼京線、中央線快速等地面路線易受影響。
3. **Taxi 可能供不應求**：建議使用 GO / S.RIDE 等 App 提前預約。
4. **考慮延後出發**：早高峰大雪時，建議彈性上班或遠距辦公。

### 攜帶物品建議

- **防滑鞋或雪靴**：東京市區路面積雪容易結冰。
- **備用乾襪子**：長時間行走容易濕透。
- **手機充電器**：查詢交通資訊需保持電量。
- **現金**：部分店家在停電時無法使用電子支付。

## 颱風 (Typhoon) 應對指南

### 計劃運休 (Planned Suspension)

JR 東日本自 2018 年起實施「計劃運休」制度：
- 颱風接近前 **12-24 小時** 發布運休預告。
- 通常 **最終電車提前至 20:00-21:00**。
- 隔日首班車可能延遲數小時。

## 地震 (Earthquake) 發生時

### 電車內遇到地震

1. **抓緊扶手**，降低重心。
2. 電車會自動緊急停車（安全措施）。
3. **聽從車掌廣播**，勿自行開門。
4. 若需疏散，從緊急出口離開。

### 車站內遇到地震

1. 遠離玻璃、販賣機、看板等可能掉落物。
2. 蹲下並保護頭部。
3. 地震結束後，遵從站員指示避難。
