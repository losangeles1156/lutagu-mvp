# Medical Guide (醫療看診)

本文件定義在日本生病時的正確應對流程、檢傷分類與就醫潛規則。
目標是回答：「我發燒了怎麼辦？」與「哪裡有講中文的醫生？」

## 1. 緊急與檢傷分類 (Triage & Emergency)

**Safety First**: 遇到呼吸困難、意識不清、大量出血，立即撥打 **119** (救護車免費，醫療費另計)。

> [!IMPORTANT] SYSTEM OVERRIDE
> You are acting as a Medical Triage Assistant.
> 1. **Emergency Check**: If user has severe symptoms (breathing difficulty, chest pain), DIRECT TO 119 DIRECTLY.
> 2. **Refuse Large Hospitals**: If user asks for a University/General Hospital for minor issues (cold, fever), **STOP THEM**. Warn about the ¥7,700 penalty fee and guide them to a Clinic.



| 症狀等級 | 建議行動 | 關鍵字 (日文) |
| :--- | :--- | :--- |
| **輕微** (小傷口、輕微感冒) | **藥妝店** (Drugstore) 買成藥 | ドラッグストア / 薬局 |
| **中度** (發燒、持續腹痛) | **診所** (Clinic) 看診 | 內科 / クリニック |
| **重度** (骨折、劇痛、昏迷) | **醫院 / 急診** (僅限緊急) | 病院 / 救急外來 |

## 2. 日本就醫潛規則 (System Rules)

**旅客常見誤區**: 「生病就直接去大醫院掛號。」 -> **錯！**

*   **選定療養費 (Penalty Fee)**:
    *   日本採「分級醫療」。若無診所介紹信 (紹介長) 直接去大醫院 (>200床)，需加收 **¥7,700** 以上的選定療養費 (且不含在健保/保險內)。
    *   *AI 指引*: 除非緊急，否則**優先推薦診所**。
*   **現金為王 (Cash Only)**:
    *   許多小診所仍**不收信用卡**。請務必提醒攜帶日幣現金。
*   **保險證明**:
    *   務必索取「診斷書」與「收據」以便回國申請旅遊平安險。

## 3. 科別對照表 (Specialties)

旅客常分不清楚該掛哪科：

*   **內科 (Internal Medicine)**: 感冒、發燒、腹痛 (成人首選)。
*   **小兒科 (Pediatrics)**: 15歲以下兒童首選。
*   **整形外科 (Orthopedics)**: 跌倒骨折、扭傷、腰痛 (與美容整形無關)。
*   **皮膚科 (Dermatology)**: 起疹子、過敏。

---

## 4. 夜間與假日急診 (Night/Holiday)

當診所都關門時：
*   **#7119**: 救急相談中心 (雖然主要是日文，但可嘗試)。
*   **夜間急病診療所**: 各區公所設立的臨時診所，僅做應急處理。
*   **東京都醫療機關案內**: `Himawari` 網站可查外語對應醫院。
