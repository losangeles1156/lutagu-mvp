# Airport Access Strategy (機場交通戰略)

本文件定義成田/羽田機場往返市區的 L4 等級決策邏輯。
目標是回答：「我該搭巴士還是電車？」與「最適合我的交通方式是什麼？」

## 1. 成田機場 (Narita Airport - NRT)

成田機場距離市區遠，交通選擇是「時間 vs 金錢 vs 體力」的博弈。

### 1.1 巴士戰略 (Bus Strategy)
*   **LCB 廉價巴士 (Low Cost Bus)** - *TYO-NRT / AIRPORT BUS*
    *   *價格*: 極神 ¥1,300 (約電車半價)。
    *   *優點*: 班次密集 (10分鐘一班)，東京站八重洲口直達，免預約。
    *   *致命缺點 (Trap)*: **行李限制極嚴**。嚴格執行「一人一件 (28-29吋)」，不可自費加量。大件樂器/滑雪板拒載。
    *   *適合*: 一人旅、行李少的省錢黨。
*   **利木津巴士 (Limousine Bus)**
    *   *價格*: 貴族級 ¥3,200+。
    *   *優點*: **Door-to-Door** (直達飯店門口)，免去拉行李轉乘地鐵之苦。每人可攜帶 2 件行李 (30kg/件)。
    *   *適合*: 帶長輩/小孩、住在新宿/澀谷大飯店、行李超多的掃貨黨。

### 1.2 電車戰略 (Train Strategy)
*   **Skyliner (京成電鐵)**
    *   *速度*: 最速王者 (36分鐘到日暮里)。
    *   *CP值*: 高 (搭配 Subway Ticket 有優惠)。但需在日暮里/上野轉乘，對新宿/澀谷旅客較麻煩。
*   **Access 特快 (Access Express)**
    *   *性質*: **通勤電車** (非對號座)。
    *   *陷阱*: 雖然便宜且直通淺草/押上，但**沒有行李架**。尖峰時刻需跟通勤族擠，拉大行李會遭到白眼。
*   **N'EX (成田特快)**
    *   *優點*: 直達新宿/橫濱/澀谷，不去上野轉車。外國人來回票 ¥5,000 很划算。
    *   *缺點*: 貴 (無優惠時)、班次較少 (30分一班)、東京站月台在地心 (地下5樓)。

## 2. 羽田機場 (Haneda Airport - HND)

距離市區近，重點在於「目的地」決定路線。

### 2.1 單軌電車 vs 京急電鐵
*   **東京單軌電車 (Tokyo Monorail)**
    *   *終點*: **濱松町站** (轉乘山手線方便)。
    *   *優勢*: 班次密集，風景好。
    *   *適合*: 住山手線沿線 (東京、有樂町、品川北側) 旅客。
*   **京急電鐵 (Keikyu Line)**
    *   *終點*: **品川站** (直通都營淺草線)。
    *   *優勢*: 便宜，且**直通運轉**至淺草/日本橋/銀座，不需轉車。
    *   *適合*: 住淺草線沿線、橫濱方向旅客。

### 2.2 深夜/早朝對策
*   **定額計程車 (Flat Rate Taxi)**
    *   羽田深夜交通匱乏，若凌晨抵達，建議預約定額計程車進市區 (約 ¥8,000-¥10,000)，比市區跳表便宜且安心。

## 3. L4 決策矩陣 (Decision Matrix)

| 需求 (User Need) | 推薦方案 (NRT) | 關鍵理由 |
| :--- | :--- | :--- |
| **極致省錢 (Budget)** | **LCB 巴士** | ¥1,300 無敵。前提是行李只有一件。 |
| **最快進城 (Speed)** | **Skyliner** | 40分鐘內進山手線圈。時間就是金錢。 |
| **帶長輩/小孩 (Family)** | **利木津巴士** | 禁止在地鐵站內做「行李重訓」。直達飯店大廳才是孝親。 |
| **住新宿/橫濱 (West Tokyo)** | **N'EX 來回票** | ¥5,000 來回，免去東京穿越戰。 |
| **住淺草/押上 (East Tokyo)** | **Access 特快** | 直達免轉乘，且票價便宜。 |
