# Fare & Ticketing Rules (票務運賃)

本文件定義東京複雜的車資計算、優惠票券適用範圍與 IC 卡規則。
目標是回答：「這樣搭多少錢？」與「我的票可以用嗎？」

## 1. 跨系統票價計算 (Cross-System Fares)

**痛點**: 東京鐵路非單一公司，轉乘不同系統會導致車資重新計算 (初乘運賃累加)。

*   **計價原則**:
    *   `Total Fare` = `Fare(System A)` + `Fare(System B)` - `Transfer Discount` (若有)。
    *   *例子*: 新宿 (JR) -> 市谷 (JR 轉地鐵) -> 飯田橋 (地鐵)。
    *   轉乘優惠: 部分 Metro 與都營地下鐵轉乘有 70 日圓折扣。
*   **直通運轉陷阱**:
    *   雖然不用下車，但車資是兩家公司分開算。
    *   *例*: 東西線直通東葉高速鐵道，車資會瞬間暴增 (東葉高速線很貴)。

## 2. 優惠票券適用與回本分析 (Discount Passes ROI)

旅客常見票券與**回本策略 (Breakeven Strategy)**：

| 票券名稱 | 適用範圍 | 回本甜蜜點 (Sweet Spot) | 陷阱 (Traps) |
| :--- | :--- | :--- | :--- |
| **Tokyo Subway Ticket** (24/48/72h) | Metro + 都營全線 | **4-5 趟/日**。平均單程 200 円計算，一天搭 4 次以上才划算。 | **JR 全線不可用** (山手線、中央線)。若行程多在 JR 沿線 (新宿-澀谷-原宿)，買這張是浪費錢。|
| **JR Tokyo Wide Pass** | JR 東日本關東地區 + 部分私鐵 | 去輕井澤/日光/河口湖來回一趟即回本 (售價 ¥15,000)。 | **市區移動**。只拿來搭山手線是暴殄天物。 |
| **Greater Tokyo Pass** | 私鐵聯盟全線 | 極難回本。需長距離跨多縣市移動才划算。 | 不包含 JR。 |

### 2.1 2025-2026 最新購票與兌換規則 (Latest Rules)

#### 🚅 JR Tokyo Wide Pass (¥15,000)
*   **購買資格**: 僅限持有非日本護照者 (Foreign Passport)。
*   **兌換地點詳解 (Exchange Nuances)**:
    1.  **護照讀取售票機 (Passport Reader Machine)**:
        *   **最推薦**。免排隊，操作僅需 5 分鐘。
        *   *識別*: 尋找貼有紅色 **"SCAN PASSPORT"** 或地球標誌的機器。
        *   *設置點*: 成田/羽田機場、東京、新宿、上野、澀谷、品川、橫濱等大站。
    2.  **JR 東日本旅行服務中心 (Travel Service Center)**:
        *   *適合*: 需要諮詢詳細行程的人。
        *   *缺點*: 成田/新宿/東京等熱門站點**排隊通常 > 1 小時**。
        *   *設置點*: 機場、東京(丸之內北口)、新宿(新南/東口)、澀谷、上野、池袋。
    3.  **綠色窗口 (Midori no Madoguchi)**:
        *   *注意*: **並非所有綠色窗口都能兌換**。僅限主要大站的指定窗口。
        *   通常會有專屬通道 (Rail Pass Counter)，勿排一般日本人的購票隊伍。

*   **使用方式**:
    *   **直接插入改札機** (磁氣票)。
    *   **劃位**: 可在綠色窗口或**指定席售票機**免費劃位 (無限次)。

#### 🚇 Tokyo Subway Ticket (QR Code Exchange)
*   **兌換機**: 尋找貼有紅色圓圈 **"Pick up your tickets"** 標誌的售票機。主要位於大站 (新宿、上野、銀座、東京)。

### 2.2 其他高 CP 值外國人車票 (Other Tourist Passes)

#### ✈️ N'EX Tokyo Round Trip Ticket (¥5,000)
*   **內容**: 成田特快 (N'EX) **來回車票** (有效期限 14 天)。
*   **CP 值**:
    *   單程原價約 ¥3,250，來回 ¥6,500。**現省 ¥1,500**。
    *   若單程去新宿/橫濱/大船，這張票無敵划算。
*   **限制**: 只能搭乘 N'EX 普通車廂指定席 (全車指定席，需先劃位)。

#### 🚅 Keisei Skyliner Discount Ticket
*   **內容**: 京成 Skyliner 單程或來回優惠票。
*   **購買**: 線上購買 (Klook/KKday/官網) 比現場買便宜約 **¥270**。
*   **Skyliner & Subway Bundle**: 包含 24/48/72h 地鐵券的套票，是進駐上野/淺草區旅客的首選。

#### 📐 Tokyu Triangle Ticket (東急三角票)
*   **內容**: 東急電鐵指定三角區域一日無限搭乘 (澀谷-自由之丘-二子玉川)。
*   **價格**: 約 ¥470 (成人)。
*   **適合**: 文青散策 (Cafe Hopping)。光是涉谷來回自由之丘就回本。

#### ⚔️ 鎌倉通票大戰 (Kamakura Pass Wars)
*   **Odakyu Enoshima-Kamakura Freepass (推薦)**:
    *   *內容*: 新宿來回 + 江之電全線。
    *   *優勢*: CP 值最高 (約 ¥1,640)，含東京出發車資。
*   **JR Kamakura-Enoshima Pass**:
    *   *內容*: 僅限鎌倉地區 JR + 江之電 + 湘南單軌。
    *   *缺點*: **不含**東京來回車資。需搭配 JR Wide Pass 才划算。


### 2.1 聰明購票策略 (Smart Ticket Strategy)
*   **短程移動 (City Center)**: 直接刷 **Suica/PASMO** 最無腦且彈性最高。不要為了省 100 日圓而被 Pass 綁死行程。
*   **私鐵特急 (Limited Express)**: 買 Pass 通常**只抵免乘車券**，特急券 (Reserved Seat) 需另外加購 (如羅曼史號、Laview)。

## 3. IC 卡使用規則與限制 (IC Card Rules & Limits)

**Suica / PASMO / ICOCA** (全國互通)

### 3.1 使用限制 (Critical Limits)
*   **跨區限制 (Area Crossing)**:
    *   *規則*: **不能**跨越不同 JR 區域 (例如：熱海=JR東日本 vs 沼津=JR東海)。
    *   *後果*: 出站時改札機門不會開。需找站務員解卡並付現。
*   **長距離限制 (Mileage Limit)**:
    *   *規則*: 雖無明文，但單程搭乘超過 **200公里** (如東京 -> 松本)，改札機可能將其視為異常。
    *   *建議*: 長距離移動直接買乘車券+特急券，不要刷 IC 卡。
*   **偏遠地區 (Remote Areas)**:
    *   *現象*: 部分鄉下小站 (如長野/新潟深山) **沒有設改札機**。
    *   *應對*: 上車領整理券，下車付現。
*   **退卡陷阱 (Refund Trap)**:
    *   *規則*: IC 卡僅能於**發行公司的站點**辦理退卡。
    *   *案例*: 在大阪買的 **ICOCA** 不能在東京退卡 (拿不回 500 円押金)。在仙台買的 Suica 也不能在東京退。
    *   *建議*: 離開該區域前退卡，或留著下次來日本用 (10年有效)。

### 3.2 餘額與手機綁定
*   **餘額不足 (Insufficient Balance)**:
    *   進站時需有最低餘額 (通常是初乘運賃，約 140 日圓)。
    *   出站餘額不足可找「精算機 (Fare Adjustment)」儲值。
*   **Mobile Suica**:
    *   強烈推薦綁定手機。
    *   *注意*: 深夜時段 (凌晨) 系統維護可能無法儲值。

### 3.3 [L5] 跨區劃位密技 (Cross-Region Booking Trick)
*   **場景**: 持有 JR Tokyo Wide Pass 但要去「河口湖」(富士回遊號)。
*   **問題**: 大月 -> 河口湖屬富士急行線，Eki-net 系統常無法直接用 Pass 劃全程座位。
*   **解法**:
    1.  Eki-net 訂票時選擇 **「不使用周遊券 (No Pass)」** 訂全程票。
    2.  抵達日本取票時，將 **Pass + 信用卡** 一起交給綠色窗口或機器。
    3.  系統會自動扣除 Pass 涵蓋範圍 (新宿-大月)，**只收差額** (大月-河口湖)。

### 3.4 [L5] 普通列車綠色車廂 (Suica Green Car System)
*   **場景**:  JR 常磐線/東海道線/總武線快速的普通列車，掛有雙層綠色車廂 (Green Car)。
*   **陷阱**: 這是**普通列車**，不是特急，但**Suica 餘額不能直接刷來坐**。
*   **正確用法**:
    1.  **上車前購買**: 在月台上的自動售票機刷卡購買「Suica Green Ticket」。(平日約 ¥1000，假日約 ¥800)。
    2.  **車內感應**: 上車後將 Suica 輕觸座位上方的讀卡機 (燈號變綠)。
    3.  **懲罰機制**: 若上車後才找乘務員補票，需多付 **¥260** 且不保證有位。

## 4. 兒童票與年齡規則 (Age Categories)

| Category | Age / Condition | Fare Rule |
| :--- | :--- | :--- |
| **Adult (大人)** | 12歲以上 (國中生以上) | 全票 (Full Fare) |
| **Child (兒童/小兒)** | 6-11歲 (小學生) | 半票 (50% Adult Fare) <br> *IC卡無條件捨去個位數 (1円單位)* <br> *實體票進位至10円單位* |
| **Toddler (幼兒)** | 1-5歲 (未就學) | **免費** (每位大人可攜帶 2 名，第 3 名起需買兒童票) |
| **Infant (乳兒)** | < 1歲 | **完全免費** |

> **學制界線**:
> *   12歲的小學生，在 3/31 畢業前仍視為「兒童」。
> *   6歲的幼兒園生，在 3/31 畢業前仍視為「幼兒」。
