# BambiGO 路線規劃使用手冊 (2026 優化版)

## 核心理念
BambiGO 的路線規劃不僅追求「最快」，更追求「最輕鬆」。我們深知在東京複雜的地下鐵系統中，一個長達 300 公尺且需要出站的轉乘（如藏前站）往往比多坐幾站地鐵更令人疲憊。

## 新版優化功能

### 1. 智能最佳建議 (Smart Route)
系統現在會綜合考慮以下因素，為您選出平衡點最佳的路線：
- **行程時間 (60%)**: 依然是核心，但不再是唯一指標。
- **轉乘便利性 (20%)**: 減少頻繁轉乘帶來的心理壓力。
- **步行距離懲罰 (15%)**: 如果轉乘步行距離超過 **150 公尺**，系統會顯著降低該路線的排名。
- **車廂擁擠度 (5%)**: 避開尖峰時段最擁擠的線路。

### 2. 轉乘地圖與設施資訊
透過新增的轉乘資料庫，系統現在能識別：
- **站內 vs 站外轉乘**: 系統會主動避開需要出站再進站的繁瑣流程。
- **垂直移動設施**: 優先推薦有扶梯或電梯的轉乘路徑（如神田站）。
- **轉乘距離**: 精確到公尺的步行預估。

### 3. 如何閱讀搜尋結果
- **[最佳建議]**: 綜合得分最高的路線。通常是時間較快且轉乘最輕鬆的選擇。
- **[最快到達]**: 適合趕時間的使用者，即使轉乘較辛苦也會被列出。
- **[最舒適]**: 優先考慮轉乘次數最少（直達優先）且最不擁擠的路線。

## 常見問題 (Q&A)

**Q: 為什麼去新宿不推薦我坐直達的山手線？**
A: 因為山手線雖然直達，但繞行距離較長。系統發現經由「神田」轉乘中央線快速只需 19 分鐘，比山手線直達快了 11 分鐘，且神田站轉乘非常近（僅 30 公尺且有扶梯），因此將其列為首選。

**Q: 為什麼某些看起來很快的轉乘路線消失了？**
A: 很有可能是因為該轉乘點（如藏前站）步行距離過長或需要出站。為了確保您的旅途輕鬆，系統在「最佳建議」中會自動降級這類「陷阱路線」。
