# LUTAGU 無障礙步行路線知識庫

## 系統概述
LUTAGU 已整合日本國土交通省的「歩行空間ネットワークデータ」，提供東京都心主要車站周邊的無障礙步行路線資訊。

## 無障礙屬性說明

### 路徑等級 (Accessibility Rank)
| 等級 | 說明 |
|------|------|
| SAA | 最佳：寬度充足、坡度平緩、設施完善 |
| SBA | 優良：大部分條件良好 |
| AAA | 良好：一般無障礙標準 |
| SBB | 需注意：部分路段可能有限制 |

### 路徑類型 (Route Structure)
| 代碼 | 類型 |
|------|------|
| 1 | 一般通道 |
| 3 | 斑馬線/人行穿越道 |
| 4 | 樓梯 (輪椅不適用) |
| 5 | 電梯 |
| 6 | 手扶梯 |

### 無障礙設施
- **電梯可達 (has_elevator_access)**: 該路段可透過電梯到達
- **導盲磚 (has_braille_tiles)**: 設有視障者導引地磚
- **有頂棚 (has_roof)**: 有遮蔽不受雨淋
- **寬度等級**: 1=<1m, 2=1-2m, 3=2-3m, 4=>3m

## 回答策略

### 當使用者詢問無障礙路線時

1. **確認區域**: 目前完整數據覆蓋「上野站周邊」
2. **說明優勢**: 99%路徑電梯可達、78%有導盲磚
3. **給予建議**: 優先選擇 S 等級路徑、避開樓梯(代碼4)
4. **補充資訊**: 有頂棚路段適合雨天

### 範例對話

**問**: 我坐輪椅，想從上野站去阿美橫町，有無障礙路線嗎？
**答**: 上野站周邊的無障礙路線非常完善！從JR上野站到阿美橫町的路徑中，99%都有電梯可達，您可以放心使用輪椅移動。建議走アメ横入口方向，該路段寬度超過3公尺，且設有導盲磚。

**問**: 上野公園附近的路好走嗎？推嬰兒車方便嗎？
**答**: 非常適合！上野公園周邊的步行路徑大多是 SAA 等級（最高級別），坡度平緩、道路寬敞。超過78%的路段設有導盲磚，路面平整，推嬰兒車完全沒問題。

**問**: 下雨天走哪條路比較好？
**答**: 建議選擇有頂棚 (has_roof) 的路段。上野站的地下通道連接良好，您可以從JR剪票口經由地下街到達阿美橫町入口，全程不會淋到雨。

## 限制與注意事項

1. **覆蓋範圍有限**: 目前僅上野站周邊有完整數據
2. **室外數據為主**: 站內詳細路線需參考各站設施資訊 (L3)
3. **數據時效**: 基於2024年整備，實際狀況可能有變動
4. **建議查證**: 重要行程建議事先向車站確認

## 技術備註 (供系統參考)

資料庫表格:
- `pedestrian_links`: 步行路徑段
- `pedestrian_nodes`: 路徑節點

關鍵欄位:
- `accessibility_rank`: 無障礙等級
- `has_elevator_access`: 電梯可達
- `has_braille_tiles`: 導盲磚
- `route_structure`: 路徑類型 (1=通道, 4=樓梯, 5=電梯)
- `floor_level`: 樓層 (0=地面, -1=B1, 0.5=夾層)
