# 使用したオープンデータ (使用的開放數據清單)

本專案開發過程參考並整合了以下日本開放數據來源：

* **公共交通オープンデータセンター (ODPT)**: [https://ckan.odpt.org/](https://ckan.odpt.org/)
    - JR 東日本の GTFS・GTFS-RT データ (JR 東日本鐵路數據)
    - 東京都交通局の都営バスの GTFS・GTFS-RT データ (都營巴士數據)
    - 東京メトロの駅・運行情報データ (東京地下鐵車站及運行資訊)
    - つくばエクスプレス (MIR) の駅データ (筑波快線車站數據)

* **GTFS データリポジトリ**: [https://gtfs-data.jp/](https://gtfs-data.jp/)
    - 東京都葛飾区「さくら」の GTFS データ (葛飾區社區巴士數據)

* **Project PLATEAU**: [https://www.mlit.go.jp/plateau/open-data/](https://www.mlit.go.jp/plateau/open-data/)
    - 3D 都市モデル (Project PLATEAU) 葛飾区 (2023 年度)
    - 東京都心 23 區站體 3D 模型數據參考

* **JMA (気象庁)**: [https://www.data.jma.go.jp/developer/xml/feed/extra.xml](https://www.data.jma.go.jp/developer/xml/feed/extra.xml)
    - 気象警報・注意報 (即時氣象警報與預警數據)

* **Open-Meteo**: [https://open-meteo.com/](https://open-meteo.com/)
    - リアルタイム天候データ (即時天氣資訊)

* **Google Maps / OpenStreetMap**:
    - 地図データ・施設情報 (地圖基礎數據與車站設施地理位置資訊)

* **VACAN**: [https://vacan.com/](https://vacan.com/)
    - リアルタイム空き情報 (即時設施人潮與空位狀態數據)

* **Jugendhackt / tokyo-metro-data**: [https://github.com/Jugendhackt/tokyo-metro-data](https://github.com/Jugendhackt/tokyo-metro-data)
    - 外部補足車站地理座標數據
